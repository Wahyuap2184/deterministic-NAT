# deterministic-NAT
metode penerjemahan alamat jaringan (NAT) yang membagi alamat IP publik dan port secara tetap kepada setiap pelanggan. Berbeda dengan NAT biasa yang memberikan port secara dinamis berdasarkan koneksi yang sedang berjalan, Deterministic NAT sudah menentukan sejak awal pelanggan mana menggunakan IP publik tertentu dan rentang port tertentu.
